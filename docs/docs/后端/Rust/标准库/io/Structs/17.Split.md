# Struct std::io::Split

```rust
pub struct Split<B> { /* private fields */ }
```

对 `BufRead` 实例的内容进行迭代的迭代器，该实例在特定字节上拆分。

通常通过在 `BufRead` 上调用 [`split`](https://www.rustwiki.org.cn/zh-CN/std/io/trait.BufRead.html#method.split) 来创建此结构体。 请参见 [`split`](https://www.rustwiki.org.cn/zh-CN/std/io/trait.BufRead.html#method.split) 的文档以获取更多详细信息。
