# Struct std::io::Repeat

```rust
pub struct Repeat { /* private fields */ }
```

一个 reader，它一遍又一遍地产生一个字节，一遍又一遍，一遍又一遍，…

通常通过调用 [`repeat()`](https://www.rustwiki.org.cn/zh-CN/std/io/fn.repeat.html) 来创建此结构体。 请参见 [`repeat()`](https://www.rustwiki.org.cn/zh-CN/std/io/fn.repeat.html) 的文档以获取更多详细信息。
