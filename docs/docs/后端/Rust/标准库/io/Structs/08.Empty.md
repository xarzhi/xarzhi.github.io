# Struct std::io::Empty

始终处于 EOF 的 reader。

通常通过调用 [`empty()`](https://www.rustwiki.org.cn/zh-CN/std/io/fn.empty.html) 来创建此结构体。 请参见 [`empty()`](https://www.rustwiki.org.cn/zh-CN/std/io/fn.empty.html) 的文档以获取更多详细信息。

```rust
#[non_exhaustive]
pub struct Empty;
```
