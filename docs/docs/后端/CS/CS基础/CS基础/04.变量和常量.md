# 变量和常量

## 1.变量

变量就是可以改变的量

声明变量

```cs
int num;
float score;
double meter;
```

声明变量并赋值

```cs
int num = 666;
float score = 95.5f;
double meter = 180.5d;
```

给变量赋值后，再修改变量的值

```cs
int num = 666;
num = 777;
```





## 2.常量

常量就是不可以改变的量，定义常量，要在修饰符前加`const` 

```c++
const int num = 666;
```

:::tip 注意

- 常量不可修改
- 声明常量必须赋值

:::
