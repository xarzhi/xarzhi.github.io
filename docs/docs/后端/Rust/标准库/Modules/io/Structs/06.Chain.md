# Struct std::io::Chain

将两个 readers 链接在一起的适配器。

这个结构体通常是通过在 reader 上调用 [`chain`](https://www.rustwiki.org.cn/zh-CN/std/io/trait.Read.html#method.chain) 来创建的。 有关更多详细信息，请参见 [`chain`](https://www.rustwiki.org.cn/zh-CN/std/io/trait.Read.html#method.chain) 的文档。

```rust
pub struct Chain<T, U> { /* private fields */ }
```





## Implementations



### impl<T, U> Chain<T, U>

#### into_inner

消耗 Chain，返回包装的 readers。

```rust
pub fn into_inner(self) -> (T, U)
```

**返回值**：返回一个元组

- 第一个元素是self的reader
- 第二个是链接的另一个reader

```rust
use std::io;
use std::io::prelude::*;
use std::fs::File;

fn main() -> io::Result<()> {
    let mut foo_file = File::open("foo.txt")?;
    let mut bar_file = File::open("bar.txt")?;

    let chain = foo_file.chain(bar_file);
    let (foo_file, bar_file) = chain.into_inner();
    Ok(())
}
```



#### get_ref

获取此 Chain 中的底层 readers 的引用。

```rust
pub fn get_ref(&self) -> (&T, &U)
```

**返回值**：返回一个元组

- 第一个元素是self的reader的**不可变借用**
- 第二个是链接的另一个reader的**不可变借用**

```rust
use std::io;
use std::io::prelude::*;
use std::fs::File;

fn main() -> io::Result<()> {
    let mut foo_file = File::open("foo.txt")?;
    let mut bar_file = File::open("bar.txt")?;

    let chain = foo_file.chain(bar_file);
    let (foo_file, bar_file) = chain.get_ref();
    Ok(())
}
```



#### get_mut

获取此 Chain 中的底层 readers 的可变引用。

```rust
pub fn get_mut(&mut self) -> (&mut T, &mut U)
```

**返回值**：返回一个元组

- 第一个元素是self的reader的**可变借用**
- 第二个是链接的另一个reader的**可变借用**

```rust
use std::io;
use std::io::prelude::*;
use std::fs::File;

fn main() -> io::Result<()> {
    let mut foo_file = File::open("foo.txt")?;
    let mut bar_file = File::open("bar.txt")?;

    let mut chain = foo_file.chain(bar_file);
    let (foo_file, bar_file) = chain.get_mut();
    Ok(())
}
```

应注意避免修改底层 readers 的内部 I/O 状态，因为这样做可能会破坏该 Chain 的内部状态。
