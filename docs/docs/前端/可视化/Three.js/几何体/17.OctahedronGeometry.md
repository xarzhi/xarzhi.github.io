# OctahedronGeometry

一个用于创建**八面体**的类。

```js
const geometry = new THREE.OctahedronGeometry(
    radius : Float,
    detail : Integer
)
```

- radius — 八面体的半径，默认值为1。
- detail — 默认值为0，将这个值设为一个大于0的数将会为它增加一些顶点，使其不再是一个八面体。
