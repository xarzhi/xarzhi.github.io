# Struct std::fs::DirBuilder

```rust
pub struct DirBuilder { /* private fields */ }
```

用于以各种方式**创建目录的构建器**。

该构建器还支持特定于平台的选项。

## 1.Implementations

### impl DirBuilder

#### new

使用所有平台的默认 `mode/security` 设置创建一组新选项，并且这些选项也是非递归的。

```rust
pub fn new() -> DirBuilder
```

**返回值**：返回一个`DirBuilder`对象

```rust
use std::fs::DirBuilder;

let builder = DirBuilder::new();
```



#### recursive

指示应递归创建目录，并创建所有父目录。 使用相同的安全性和权限设置创建不存在的父级。

此选项默认为 `false`。

```rust
pub fn recursive(&mut self, recursive: bool) -> &mut Self
```

**参数**：

- **recursive**：指定创建目录时是否可以递归创建

**返回值**：返回自身

```rust
use std::fs::DirBuilder;

let mut builder = DirBuilder::new();
builder.recursive(true);
```



#### create

使用在此构建器中配置的选项来创建指定的目录。

如果目录已经存在，必须调用`recursive(true)`，否则会发生错误

```rust
pub fn create<P: AsRef<Path>>(&self, path: P) -> Result<()>
```

**参数**：

- **path**：创建目录的路径

**返回值**：返回一个`Result`，若成功则返回空，否则返回失败信息

```rust
use std::fs::DirBuilder;

fn main() {
    let path = "./tmp/foo/bar/baz";
    DirBuilder::new().recursive(true).create(path).unwrap();
}
```
