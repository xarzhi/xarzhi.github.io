# FileTimes

> Struct std::fs::FileTimes

文件上各种时间戳的表示



## 1.方法

### new()

创建一个没有设置时间的新 `FileTimes`。

```rust
pub fn new() -> Self
```

返回值：返回一个新的`FileTimes`

```rust
use std::fs::FileTimes;

fn main() {
    let t = FileTimes::new();
    println!("{:#?}", t);

	/*
		FileTimes(
			FileTimes {
				accessed: None,
				modified: None,
				created: None,
			},
		)
	 */
}
```



### set_accessed()

设置文件的最后访问时间。

```rust
pub fn set_accessed(self, t: SystemTime) -> Self
```

参数：

- t：系统时钟的度量[SystemTime](https://www.rustwiki.org.cn/zh-CN/std/time/struct.SystemTime.html#)

返回值：返回自身

```rust
use std::fs::FileTimes;
use std::time::SystemTime;
fn main() {
    let t = FileTimes::new();
	t.set_accessed(SystemTime::now());
    println!("{:#?}", t);
}

```
