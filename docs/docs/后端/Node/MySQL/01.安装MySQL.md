# 安装MySQL

原文来自CSDN：[MySQL8.0最新最全下载及安装配置教程（保姆级）_mysql8.0安装教程-CSDN博客](https://blog.csdn.net/qq_65771647/article/details/147590517)

## 一、卸载旧版本MySQL

如果各位小伙伴之前有下载过别的MySQL版本，在更新前一定要确保自己删除干净旧版本的所有痕迹，包括注册表、缓存等内容。

### 1.清除安装残留

   **(1)** 在搜索栏搜索控制面板，点击进入。

![img](https://gitee.com/xarzhi/picture/raw/master/img/ad6b21580f454c71a71332c5d15704b5.png)

  **(2)** 进入后点击程序下方的卸载程序，找到自己之前下载过的MySQL版本，右键卸载即可。

![img](https://gitee.com/xarzhi/picture/raw/master/img/6dd8341abc6d47dc937e8cb66d0e34a7.png)

![img](https://gitee.com/xarzhi/picture/raw/master/img/9422c7abefa840a0b481c6c21979248d.png)

### 2.[删除目录](https://so.csdn.net/so/search?q=删除目录&spm=1001.2101.3001.7020)

   **(1)** 找到之前自己下载过的MySQL版本路径下的MySQL文件夹删除（如果之前下载时默认安装在C盘则在Program Files中，如果修改过路径就在其他盘就找到当时设置的安装路径下）

![img](https://gitee.com/xarzhi/picture/raw/master/img/bd0bce1a3d154826ae188355b5c88867.png)

  **(2)** C盘中查看栏，勾选“隐藏的项目”找到ProgramData文件夹，点开，删除MySQL文件夹。

![img](https://gitee.com/xarzhi/picture/raw/master/img/55a47e41c6d541899b5b90113e799926.png)

![img](https://gitee.com/xarzhi/picture/raw/master/img/c126a4e55b494e1bb1c7019bf0b5e4e4.png)

### 3.删除注册表

  **(1)** win+R键输入 regedit 运行，进入[注册表编辑器](https://so.csdn.net/so/search?q=注册表编辑器&spm=1001.2101.3001.7020)

![img](https://gitee.com/xarzhi/picture/raw/master/img/e5778b555a1b4d50bb9dd9e3b15cdb07.png)

  **(2)** 按下图路径找到MySQL，进一步右键删除MySQL注册表信息。（如果在该路径中没有找到MySQL说明注册表信息已清除，可以跳过这一步）

```cpp
计算机\HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Services\EventLog\Application\MySQL
运行项目并下载源码cpp
运行
```

![img](https://gitee.com/xarzhi/picture/raw/master/img/a7e5b5230c774d8a89bdde8829963214.png)

### 4.[删除MySQL服务](https://so.csdn.net/so/search?q=删除MySQL服务&spm=1001.2101.3001.7020)

  **(1)** 以管理员身份运行命令提示符

![img](https://gitee.com/xarzhi/picture/raw/master/img/878a981bc02d460a89cf4e3dd23e0bbd.png)

  **(2) 输入**sc delete MySQL80，MySQL80就是你之前下载过的MySQL定义的名字，回车删除即可。PS：我删除的时候忘记截图了，删除成功应该会提示[SC] DeleteService 成功。

![img](https://gitee.com/xarzhi/picture/raw/master/img/7a18466aa2f448ba9dabd013233dabff.png)

## 二、MySQL下载

### 1.下载安装包

  **(1)** 点击链接进入MySQL官方下载地址https://www.mysql.com/cn/downloads/

进入官网后点击下载模块，下滑找到MySQL社区版下载。

![img](https://gitee.com/xarzhi/picture/raw/master/img/11eeecc960f549888f6ac554fda833e0.png)

![img](https://gitee.com/xarzhi/picture/raw/master/img/a4ad0d46a4c44843b9065ac6ba6a7178.png)

  **(2) 选择Windows版本**

![img](https://gitee.com/xarzhi/picture/raw/master/img/fa3b5ddf17bf4b91bcdf3edbcd507c50.png)

  **(3)选择离线进行安装**

注意：在这里MSI Instealler有两个，第一个（大小2.1M）是通过联网在线安装，会在线下载安装包；第二个（大小353.7M）是离线安装。我们选第二个，下载到本地后进行安装。

![img](https://gitee.com/xarzhi/picture/raw/master/img/65e3d7dfaa0c471ba9dd1bb48c9cdaee.png)

![img](https://gitee.com/xarzhi/picture/raw/master/img/efb0299a56054f559f9123917debab2c.png)

  **(2)**选择自定义custom，接着下一步

![img](https://gitee.com/xarzhi/picture/raw/master/img/46c561db61b84e0d8b604ff7598a297c.png)

  **(3)**将MySQL Server 逐层展开，添加到右边（鼠标点击绿色箭头），如下图

![img](https://gitee.com/xarzhi/picture/raw/master/img/6bc0c248f12444459c18473b958b4c30.png)

  **(4)加载100%后点击Execute,之后一直点击Next即可**![img](https://gitee.com/xarzhi/picture/raw/master/img/ba94f3669e1541979710273bcd3c2a92.png)

## 三、环境配置

  **(1)**找到你安装好的MySQL的bin目录下，如果默认安装路径的话，直接复制即可

```cpp
C:\Program Files\MySQL\MySQL Server 8.0\bin
运行项目并下载源码cpp
运行
```

![img](https://gitee.com/xarzhi/picture/raw/master/img/92d98cb12155407b937d67d0c1468a19.png)

  **(2)**右击此电脑，选择属性，下滑找到高级系统设置，选择环境变量。

![img](https://gitee.com/xarzhi/picture/raw/master/img/03a0b250a28c4d589412e3a1404f2b9a.png)

  **(3)**在系统变量中下滑找到Path,双击打开。

![img](https://gitee.com/xarzhi/picture/raw/master/img/3d9dead47b824792b0da1c26c668c4a8.png)

  **(4)**如果之前有配置过旧版本的MySQL环境变量，先将其删除。如果没有，跳过此步骤即可。

![img](https://gitee.com/xarzhi/picture/raw/master/img/27fd3b17bd4b4a40811a206bd63ab1c1.png)

  **(5)**首先点击新建，将之前复制好的MySQL的bin目录复制粘贴进去，再点击上移将其放在合适位置（目的就是mysql可能常用，上移使其优先级高一点，响应快，不过大体没啥影响，随意即可）。

![img](https://gitee.com/xarzhi/picture/raw/master/img/0c3cb96ff81b41ca80468284490aee12.png)

  **(6)**切记，逐层点击确定关闭窗口！！！

![img](https://gitee.com/xarzhi/picture/raw/master/img/64fbf227dacd4e508010dc7e725d2e0a.png)

## 四、验证是否安装成功

### 1.命令提示符窗口进行验证

  **(1)**键盘使用 win+R键打开命令命令提示符，输入cmd，回车确认即可

![img](https://gitee.com/xarzhi/picture/raw/master/img/d56f9593259746fc90dcbb1cfbf61ccd.png)

  **(2)**输入mysql -h localhost -u root -p登录数据库，再输入数据库密码，查看信息。

![img](https://gitee.com/xarzhi/picture/raw/master/img/dd9ddb80abdc4afabcd4ec32eb673cbf.png)

  **(3)**输入status 命令可以查看MySQL的版本信息，说明安装成功。

![img](https://gitee.com/xarzhi/picture/raw/master/img/983916cb758c455983111f6dbf23eacc.png)

### 2.控制台验证

  **(1)**在 “开始” 菜单栏找到并打开 MySQL 8.0 Command Line Client 程序。

![img](https://gitee.com/xarzhi/picture/raw/master/img/0d441903f01a4c2cac7eafdebbfd83fe.png)

  **(2)**输入数据库密码后按回车键，弹出当前MySQL的服务状态。当前MySQL服务器的版本为 8.0.42 MySQL Community Server - GPL，表明安装成功。

![img](https://gitee.com/xarzhi/picture/raw/master/img/936019c8bebb469b999aa85b079d4f86.png)

至此，MySQL成功安装。如果电脑之前有下载navicate，需要连接继续看下面就可以，如果未下载过，到此结束即可。

## 五、Navicat连接

  **(1)**打开Navicat Premium,点击左上角连接，然后选择MySQL。

![img](https://gitee.com/xarzhi/picture/raw/master/img/39857a7ce09746b3bc6fe3b18a88c422.png)

  **(2)**输入自己的数据库名，如果不确定可以win+R，输入services.msc，回车进入服务查看自己的数据库名。如下图所示，可以看到我的是MySQL80。

![img](https://gitee.com/xarzhi/picture/raw/master/img/45dcf74f9c7a41cbbe1b4ef430c3f77e.png)

![img](https://gitee.com/xarzhi/picture/raw/master/img/dcc8225d778e41ccb6855857b2966a2c.png)


![img](https://gitee.com/xarzhi/picture/raw/master/img/949ecfb388c440d39c1ad5ae0bf5b564.png)

![img](https://gitee.com/xarzhi/picture/raw/master/img/6fdc14aa6c684361b70bec2d6b5bb0e4.png)

   **(3)**部分同学可能会出现连接不成功的问题，检查一下自己的MySQL是否属于开机自动启动。步骤同上，win+R输入services.msc，回车进入服务，下滑找到数据库MySQL80,双击打开，设置启动类型为自动，并确认。
