# sealed

`sealed`是个关键字，可以修饰类，也可以用来修饰方法

## 1.密封类

被`sealed`修饰的类叫做密封类，密封类**不可以被继承**

```cs
sealed class Father { }

class Son : Father { }
```

## 2.密封方法

被`sealed`修饰的方法叫做密封方法，密封方法**不能被重写**

并且只有被override重写的方法才能被蜜蜂

```cs {7,12}
class Father
{
    public virtual void speak() { }
}

class Son : Father
{
    sealed public override void speak() { }
}

class grandson : Son
{
    public override void speak() { }
}
```
