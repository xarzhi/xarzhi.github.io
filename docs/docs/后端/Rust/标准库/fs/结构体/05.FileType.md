# FileType

> Struct std::fs::FileType

表示文件类型的结构体，每个文件类型都有访问器。 通过 [`Metadata::file_type`](./06.Metadata#file-type) 方法返回。

## 1.Implementations

### impl FileType

#### is_dir

判断此文件类型是否为目录（文件夹）

```rust
pub fn is_dir(&self) -> bool
```

**返回值**：根据文件类型是否为目录，返回bool值



#### is_file

根据文件类型是否为文件

```rust
pub fn is_file(&self) -> bool
```

**返回值**：根据文件类型是否为文件，返回bool值

:::tip

当目标只是读取 (或写入) 源时，可以读取 (或写入) 最可靠的测试源方法是打开它。 例如，仅使用 `is_file` 才能中断类似 Unix 的系统上的工作流，例如 `diff <( prog_a )`。 有关更多信息，请参见 [`File::open`](https://www.rustwiki.org.cn/zh-CN/std/fs/struct.File.html#method.open) 或 [`OpenOptions::open`](https://www.rustwiki.org.cn/zh-CN/std/fs/struct.OpenOptions.html#method.open)。

:::



#### is_symlink

判断此文件类型是否为软链接

```rust
pub fn is_symlink(&self) -> bool
```

**返回值**：根据文件类型是否为软链接，返回bool值

:::tip

需要使用 [`fs::symlink_metadata`](https://www.rustwiki.org.cn/zh-CN/std/fs/fn.symlink_metadata.html) 函数而不是 [`fs::metadata`](https://www.rustwiki.org.cn/zh-CN/std/fs/fn.metadata.html) 函数来检索底层 [`Metadata`](https://www.rustwiki.org.cn/zh-CN/std/fs/struct.Metadata.html) 结构。 [`fs::metadata`](https://www.rustwiki.org.cn/zh-CN/std/fs/fn.metadata.html) 函数遵循符号链接，因此 [`is_symlink`](https://www.rustwiki.org.cn/zh-CN/std/fs/struct.FileType.html#method.is_symlink) 将始终为目标文件返回 `false`。

:::
