# Enum std::io::SeekFrom

列举可能在 I/O 对象中进行搜索的方法。

```rust
pub enum SeekFrom {
    Start(u64),
    End(i64),
    Current(i64),
}
```

`SeekFrom`实现了`Seek` Trait





## Variants

### Start(u64)

将偏移量设置为提供的字节数。

从第 n 个字节开始



### End(i64)

将偏移量设置为此对象的大小加上指定的字节数。

可以在 object 的末尾之外进行 seek，但在字节之前进行 seek 是错误的 0.

从文件末尾 ± n

### Current(i64)

将偏移量设置为当前位置加上指定的字节数。

可以在 object 的末尾之外进行 seek，但在字节之前进行 seek 是错误的 0.

从当前位置 ± n
